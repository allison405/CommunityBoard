# Web Development Project 1 - *Studio Ghibli Movie Reviews*

Submitted by: **Allison Lee**

This web app: **Provides users with a space to check out some of Hayao Miyazaki's renowned Studio Ghibli Films. You can easily take a look at the movie, its rating and look at more in depth reviews about the film on the IMDb website.**

## Notes

During the build of this website, it was difficult figuring out how to space/align each movie card, but eventually worked out nicely with justifying content with space in between. 
The newly learned props feature made coding out the cards incredibly efficient, simply requiring the set up of a basic framework, and then being able to code in the specialized content like images, title, ratings, etc. This also allowed the css process to be simpler, as styling would be the same across all cards.
